This project uses **Plop** to auto-generate:

- Controller

- Module  
- Model  
- DTOs  
- Folder structure  

To generate a new module, simply run:

```bash

```

## Project Setup

```bash
yarn install
```

## Compile and Run the Project

```bash
# development
yarn run start

# watch mode
yarn run start:dev

# production mode
yarn run start:prod
```

## Run Tests
